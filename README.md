# Penetration Testing - IRAT Preparation

Interactive IRAT preparation quiz covering Vulnerability Scanning, CVSS, and Penetration Testing. 45 questions across Easy, Medium, Hard, and Scenario difficulty levels.

## Topics Covered

- Vulnerability Scanning (types, scan methods, discovery, fingerprinting)
- Common Vulnerability Scoring System (CVSS)
- Penetration Testing (methodology, scope, reporting)

## Sources

- Introduction to Vulnerability Scanning module
- CVSS Definition by TechTarget
- Penetration Testing Guidance by NCSC
- What is a Pen Test by Rapid7 Whiteboard Wednesday

## Features

- 45 questions across 4 difficulty levels
- Instant feedback with explanations
- Score tracker and progress bar
- Reset and retry
